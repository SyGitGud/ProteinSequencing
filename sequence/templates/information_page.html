About the Product

The web application is developed by the members of Team 5 for COMP 496: Senior Project II. This product uses an AI model with a K Nearest-Neighbor algorithm. This locally hosted website was created using Django, CSS, HTML, and SQLite3.


Vision

The product's vision is to create an informative introduction to working with models that predict complete protein scaffolds from incomplete protein scaffolds by using a clean and detailed interface.


How does the model work?

The model uses the k-nearest neighbor (KNN) algorithm to take an incomplete scaffold, fill in the scaffold gaps, and output the complete scaffold and training data.


What is the purpose of the model?

The purpose of the model is to predict and complete protein sequences that are only partially known to researchers.

How to use this product?

  • Follow the README instructions for setup
  • Enter the incomplete scaffold in the textbox.
  • Click the submit button.
  • The complete scaffold, output accuracy, and training sequence will be displayed to the corresponding textfield.
