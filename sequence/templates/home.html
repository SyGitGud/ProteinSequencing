Welcome to the Protein Sequencing page. This project developed by Team 5 is design to create an easy and navigable interface for users seeking to find a complete protein sequence. Using FASTA protein scaffolds format, enter an incomplete protein scaffold in the input, and a complete protein scaffold will be returned. This is done in the background using a Knn model that is tuned, tweaked, and maintained by Team 5.
